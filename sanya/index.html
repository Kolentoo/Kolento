                                                    # 春秋旅游上海飞海南三亚自由行5天4晚机票红树林亚特兰蒂斯酒店

                                                    订单号: 1934891113478489081
                                                    套餐类型： 晚航班-4晚万达嘉华
                                                    出行日期： 2023-08-26 - 2023-08-30

                                                    注意防晒，据说一天能黑2个度

                                                    航班信息仅供参考，对应的航班时间均为当地时间，存在航班调整的可能（包括航班号、航班时刻的变化等）

                                                    # 酒店信息

                                                    酒店名称： 三亚海棠湾万达嘉华度假酒店
                                                    酒店距离三亚凤凰机场40.6公里，如打车约 140-150元

                                                    房型双早购物
                                                    园景房含双早三亚国际免税城

                                                    购物： 据说可以使用小程序CDF海南可以直接购买。

                                                    # 旅游费用包含

                                                    嘉华酒店入住礼遇

                                                    1. 每日免费迷你吧
                                                    2. RUI X网红餐厅免费咖啡2杯
                                                    3. 每日儿童营亲子活动
                                                    4. 每日泳池泡沫派对
                                                    5. 海棠湾国际免税城免费穿梭巴士9.5折和150元代金券
                                                    6. 亚特兰蒂斯C秀/水世界/水族馆门票电子折扣券（客人到酒店前台和前台问询，二维码扫描有相对应的折扣）
                                                    7. 槟榔谷5A门票2张（免费项目，客人到酒店前台告知前台，前台会给予安排）
                                                    8. 海昌不夜城小黄鱼馆潜水门票2张
                                                    9. 海昌欢乐剧场表演门票2张（价值128元）
                                                    10. Reiki SPA满1000减200元代金券

                                                    # 亚特兰蒂斯水世界

                                                    景点介绍详情open in new window
                                                    水世界里玩水的项目很多
                                                    主要分为： 水上世界（夜场16:00-21:00）、水族馆
                                                    地址： 三亚市海棠区海棠北路与风塘路交叉路口往东约200米
                                                    开放时间； 10:00-21:00 （20:00停止售票，20:00停止入园）
                                                    官方电话： 0898-88986666
                                                    优惠政策： 儿童：身高0.9米(不含)以下，免费；身高0.9米(含)-1.2米(含)或年龄2周岁(含)-4周岁(含)，优惠；身高1.21米(含)-1.5米(含)或年龄5周岁(含)-11周岁(含)，优惠

                                                    门票组合有很多种，可以买亲子票，家庭票等

                                                    # 蜈支洲岛

                                                    适合潜水和各种水上项目
                                                    景点介绍详情open in new window
                                                    一般购买门票包含往返船票即可
                                                    景点地址： 海南省三亚市海棠湾镇蜈支洲岛地图
                                                    开放时间： 08:00－17:30，最晚上岛时间：16:00 ，最晚下岛时间：18:00
                                                    注意： 涉水项目请自带泳衣

                                                    # 亚龙湾热带天堂森林公园

                                                    景点介绍详情open in new window
                                                    景点地址： 海南省三亚市亚龙湾区湾路665号亚龙湾国家旅游度假区。
                                                    开放时间： 全年 08:00-18:00。
                                                    整个过程： 取门票——坐览车——过江龙索桥——雨林魔幻电影院——玻璃栈道——雨林飞漂——兰花谷——坐游缆车下山---离开景区
                                                    推荐项目： 过江龙索道 玻璃栈道 雨林飞漂
                                                    这一带好像都算是亚龙湾国家旅游度假区

                                                    # 推荐行程

                                                    以下仅为推荐行程
                                                    请根据实际情况自由调整安排

                                                    门票购买： 携程/途牛/飞猪/美团

                                                    # 第一天 8月26日

                                                    约21:35降落，22.30前到达酒店
                                                    美食/小吃夜市： 林旺夜市广场 距离6公里左右，口碑还行，地方不大（建议游玩时间：15-30分钟）
                                                    或者其他距离酒店不远的美食小吃（海棠68环球美食商业），可以大众点评附近搜一下
                                                    注意事项： 最好提前一天买好第二天要去的景区门票

                                                    # 第二天 8月27日

                                                    景区： 亚特兰蒂斯水世界+水族馆（需门票，可选择白天或者夜场，建议游玩时间：4-5小时 + 2小时）
                                                    美食： 后海村（小吃街，美食，网红店铺，建议游玩时间：1-3小时）
                                                    注意事项： 泳衣、防晒

                                                    # 第三天 8月28日

                                                    景区： 蜈支洲岛旅游风景区 （需门票+往返船票，简单游玩观景3小时/游玩项目4-6小时/充分体验3-6天）
                                                    美食： 海棠68环球美食商业
                                                    景区包含的项目： 飞龙，潜水，拖伞，摩托艇等

                                                    # 第四天 8月29日

                                                    景区： 热带天堂森林公园 （需门票，建议游玩时间：3到5个小时）
                                                    比较有代表性的包括雨林滑索、鸟巢度假村、玻璃栈道和雨林飞漂等

                                                    从 热带天堂森林公园 打车过去20公里
                                                    景区： 大东海旅游区（不需门票，水上运动，如潜水、帆板、摩托艇等。海湾周边有许多酒店、餐厅和酒吧，供游客休息和享受美食，建议游玩时间：2-3小时）
                                                    景区： 鹿回头风景区（风景、特色餐饮和手工艺品店） （建议游玩时间：2-3小时）
                                                    鹿回头风景区：通过景区微信／支付宝小程序“0元购票”链接，进行实名预约，预约成功后持本人身份证件即可核销入园

                                                    # 第五天 8月30日

                                                    景区： 三亚海昌梦幻不夜城（游乐场，不需门票，项目游玩收费，建议游玩时间：2-5小时）
                                                    购物： 中免集团三亚免税店，距离很近
                                                    最后一天的景点可以选择距离酒店近一些，
                                                    即便退房后可以把行李寄存酒店，玩好回来拿，吃好晚餐去机场。
                                                    出发去机场时间： 19：30
                                                    离开航班时间： 22:30 三亚凤凰
                                                    降落浦东T2： 01：55

                                                    # 携带物品

                                                    • 身份证

                                                    • 衣物：根据季节和天气情况选择合适的衣物，包括防晒衣、泳衣、夏季服装、太阳帽、墨镜等。

                                                    • 防晒用品：防晒霜、遮阳伞、遮阳帽等，以保护皮肤免受阳光伤害。

                                                    • 鞋类：舒适的步行鞋、凉鞋或拖鞋，适合海滩或户外活动。

                                                    • 洗漱用品：牙刷、牙膏、洗发水、沐浴露、毛巾等个人卫生用品。

                                                    • 药品：常用药品、创可贴、感冒药等基本的急救药品。

                                                    • 电子设备：手机、数据线、充电器、移动电源、直播支架、散热器等。

                                                    • 游泳用具：泳衣 泳镜

                                                    # 注意事项

                                                    • 注意不要在暗礁上或者潮汐比较急的地方游泳，防止发生意外。
                                                    • 潜水时不要触碰珊瑚和鱼类，以免造成伤害。
                                                    • 注意防晒和保湿，因为海边的阳光和海风可能会让皮肤变得干燥。
                                                    • 在购买海鲜时，要注意价格和质量，避免被坑
                                                    • 多喝水防中暑
