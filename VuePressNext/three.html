                      # vuepress2插件

                      # 评论插件

                      vuepress-plugin-comment2open in new window
                      可以用于vuepress2 的一款评论留言插件。
                      注意： 该插件在vuepress默认主题下，打包的时候会报错，推荐在使用VuePress Theme Hope主题的时候使用。

                      # 安装

                      npm install vuepress-plugin-comment2 -d --save

                      # 选项配置

                      import { commentPlugin } from "vuepress-plugin-comment2";

// .vuepress/config.js
export default {
  plugins: [
    commentPlugin({
      provider: "Artalk", // Artalk | Giscus | Waline | Twikoo

      // 在这里放置其他选项
      // ...
    }),
  ],
};

                      # 参数解释

                      • provider 有四种参数，分别为 Artalk | Giscus | Waline | Twikoo
                        Giscus 是针对于程序员开发者使用，评论留言需要登录 github
                        Waline 是针对于大众用户，适合所有人。

                      # LeanCloud 设置 (数据库)

                      1. 登录open in new window 或 注册open in new window LeanCloud 国际版 并进入 控制台open in new window

                      2. 进行邮箱等认证

                      3. 点击左上角 创建应用open in new window 并起一个你喜欢的名字

                      4. 进入应用后，选择 凭证 ，保存 APP ID,APP Key 和 Master Key

                      5. 这里推荐使用 国际版open in new window,因为默认的国内版需要备案。

                      # Vercel 部署

                      1. 点击这个连接 服务部署open in new window

                      2. 进入后，先进行注册登录，或者使用 github快捷登录。

                      3. 输入一个你喜欢的 Vercel 项目名称并点击 Create 继续

                      4. 此时 Vercel 会基于 Waline 模板帮助你新建并初始化仓库，仓库名为你之前输入的项目名

                      5. 一两分钟后，满屏的烟花会庆祝你部署成功。此时点击 Go to Dashboard 可以跳转到应用的控制台

                      6. 点击顶部的Settings - Environment Variables进入环境变量配置页，并配置三个环境变量 LEAN_ID, LEAN_KEY 和 LEAN_MASTER_KEY,它们的值分别对应上一步在 LeanCloud 中获得的 APP ID, APP KEY, Master Key。

                      7. 此时会跳转到 Overview 界面开始部署，等待片刻后 STATUS 会变成 Ready。此时请点击 Visit ，即可跳转到部署好的网站地址，此地址即为你的服务端地址。

                      # 使用插件

                      在插件选项中设置 provider: "Waline"，同时设置服务端地址 serverURL 为上一步获取到的值
                      配置如下

                          commentPlugin({
      provider: "Waline", //评论服务提供者。
      serverURL:'https://kolento-xxxxxxx.vercel.app/'
    }),

                      此时，将<CommentService>组件放置在你网站中合适的位置 (通常是页面的底部)，即可使用 Waline 评论功能
                      如果已经使用了hope 主题 ，就不需要再防止这个标签了。

                      # 评论管理

                      1. 部署完成后，请访问 <serverURL>/ui/register 进行注册。首个注册的人会被设定成管理员。
                      2. 管理员登陆后，即可看到评论管理界面。在这里可以修改、标记或删除评论。
                      3. 用户也可通过评论框注册账号，登陆后会跳转到自己的档案页。
