                      # 个人项目

                      # 二次元番剧

                      # 简介

                      这是一个用于查阅影视、动漫、电影、小说剧集信息的微信小程序，由本人开发。
                      本项目只作为学习娱乐使用。
                      目前由于服务器到期，已经停止维护。仅作为项目记录留念。

                      # 图片介绍

                      # 内容

                      # 主要包含以下内容

                      1.动漫（数据来源bangumi动漫网，通过爬虫每日爬取数据存入数据库）
                      2.电影（数据来源douban网，通过服务端转发豆瓣公开接口）
                      3.电视剧（数据来源douban网，通过服务端转发豆瓣公开接口）
                      4.小说（数据来源bangumi动漫网，通过爬虫每日爬取数据存入数据库）

                      # 功能介绍

                      1.搜索功能：通过首页的搜索栏目搜索需要找到的剧集信息。
                      2.筛选功能：在动漫、小说栏目可以通过时间、年份和剧集类型筛选信息。
                      3.评价留言功能：可以对剧集进行评论，也可以对小程序本身进行留言。
                      4.收藏功能：收藏自己喜爱的剧集。
                      5.登录功能：在个人主页点击登录，获取微信头像、名字、性别进行登录，会把新用户存储到数据库。
                      6.分享功能，将小程序分享给其他人。

                      # 技术架构

                      前端：微信小程序源生开发
                      后端：nodejs开发，主要开发一些接口（搜索查询、登录、评论留言、收藏等功能，转发一些接口）
                      服务端：linux
                      数据库：mysql（用于数据存储）
                      数据来源：puppetter （每日定期爬虫爬取数据存储到数据库）

                      # 后台管理系统模板

                      Vue3-Ant-design
                      这是一个从零开始开发的 基于Vue3.x和Ant-design-Vue的后台管理系统模板。

                      # 聊天室

                      # 项目介绍

                      基于 Vue、Electron与websocket 进行开发的聊天室软件。
                      功能：

                      • 登录功能、注册名称功能
                      • 可以多人在一个聊天室内进行文字聊天、发送表情包等功能。
                      • 进入聊天室提醒功能。
