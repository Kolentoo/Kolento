                              # 标签页

                              当打开多个页面进行后台系统操作时，通过标签页切换页面，
                              并且保留原先的操作记录，这也是比较常见的功能。

                              # 新建标签页

                              在 components 文件夹下新建标签页组件。
                              使用的是 ant 内置的tab标签组件。

                              <template>
  <div class="nav-box">
    <a-tabs size="small" v-model:activeKey="activeKey" type="editable-card" @edit="onEdit" @tabClick="chooseNav" hide-add>
      <a-tab-pane v-for="item in tabBox.tabList" :key="item.key" :tab="item.title" :closable="item.closable" >
      </a-tab-pane>
    </a-tabs>
  </div>
</template>
                              • activeKey 代表当前选中的key
                              • onEdit 添加/删除 标签页事件
                              • chooseNav 为标签点击事件
                              • tabBox.tabList 为标签页的数据，是一个数组
                              • closable 表示标签是否有关闭按钮

                              # 引入组件

                              1. 在 main.vue 中引入标签页组件
                              2. 改写原先的content内容部分，添加 keep-alive进行组件缓存，用component的形式进行组件切换。
                              <template>
  <div class="main">
    <div :class="['side-box',smallSide?'small-side':'']">
      <div class="logo">
        <p class="title">Kolento</p>
      </div>
      <sideBar />
    </div>
    <div :class="['content-box',smallSide?'big-content':'']">
      <topBar />
      <!-- navBar是标签页 -->
      <navBar />
      <div class="content">
        <router-view v-slot="{ Component }">
          <keep-alive>
            <component :is="Component" />
          </keep-alive>
        </router-view>
      </div>
      
    </div>
  </div>
</template>

                              # 添加数据存储

                              主要是存储标签页的数据，存储与pinia中。

                              1. 新建一个tabStore用于存储数据
                              2. 添加persist数据持久化
                              3. 在state中存储 tabLsit 存放标签页数据
                              4. 添加 addTab 和 removeTab 方法来添加删除标签页
                              // 标签页信息
export const tabStore = defineStore('tab',{
  persist: {
    enabled: true,//开启数据持久化
    strategies: [
      {
        key: 'tab',//给一个要保存的名称
        storage: sessionStorage,//sessionStorage / localStorage 存储方式
      }
    ]
  },
  state:()=>{
    return {
      tabList:[
        {    
          path:'/',
          title: '首页',
          key:'home',
          closable:false,
        }
      ],
    }
  },
  actions:{
    addTab(state,tab){
      console.log('add state',state)
      console.log('add tab',tab)
      // 判断是否已经存在该标签
      const isSome = state.some((item)=>{
        return item.path==tab.path
      })
      if(!isSome){
        state.push(tab);
      }
    },
    removeTab(state,tab){
      console.log('remove state',state)
      console.log('remove tab',tab)
      let cindex = 0;
      state.filter((item,idx)=>{
        if(item.key==tab){
          cindex=idx
        }
      });
      state.splice(cindex,1);
    }
  }
})

                              # 具体步骤

                              1. 设置默认选中key，获取pinia仓库中标签页数据的第一个进行默认选中。
                              2. 通过路由跳转，获取当前路由的信息，包括之前添加配置的meta信息，把数据传递到 pinia 数据仓库的添加标签方法中。
                              3. 添加关闭标签页事件，这里关闭主要做两个事情。
                              • 点击关闭标签页，删除当前的标签，这个通过 pinia 中的方法 tabBox.removeTab(tabBox.tabList,targetKey);实现。
                              • 点击关闭，将原来选中的key修改为前一个标签页，并且路由也跳转到前一个标签页对应的页面。
                              1. 点击标签页chooseNav，进行对应的路由跳转
                              2. 路由监听，在路由发生变化时，添加对应的标签页

                              navBar.vue

                              <template>
  <div class="nav-box">
    <a-tabs size="small" v-model:activeKey="activeKey" type="editable-card" @edit="onEdit" @tabClick="chooseNav" hide-add>
      <a-tab-pane v-for="item in tabBox.tabList" :key="item.key" :tab="item.title" :closable="item.closable" >
      </a-tab-pane>
    </a-tabs>
  </div>
</template>

<script setup>
import { ref,watch } from 'vue';
import {tabStore} from '../stores/index'
import {useRoute,useRouter} from 'vue-router';
const route = useRoute();
const router = useRouter();
let tabBox = tabStore();

// 设置默认选中tab
const activeKey = ref(tabBox.tabList[0].key);

// 添加标签页
const add = () => {
  let newTab = {
    path:route.path,
    title: route.meta.title,
    key: route.meta.name,
  }
  tabBox.addTab(tabBox.tabList,newTab);
};

// 关闭标签页【处理关闭标签页后标签页的tab跳转和页面跳转】
const remove = targetKey => {
  let lastIndex = 0;
  tabBox.tabList.forEach((list, index) => {
    if (list.key === targetKey) {
      lastIndex = index - 1;
    }
  });
  if (tabBox.tabList.length && activeKey.value === targetKey) {
    if (lastIndex >= 0) {
      activeKey.value = tabBox.tabList[lastIndex].key;
    } else {
      activeKey.value = tabBox.tabList[0].key;
    }
    router.push({name:activeKey.value})
  }
};

// 操作标签页
const onEdit = (targetKey, action) => {
  if (action === 'add') {
    add();
  } else {
    remove(targetKey);
    tabBox.removeTab(tabBox.tabList,targetKey);
  }
};

let chooseNav=(item)=>{
  console.log('item',item);
  router.push({name:item});
}

watch(()=>route.path,()=>{
  // 路由变化时，判断是否需要添加标签页
  console.log('监听路由变化');
  activeKey.value=route.meta.name;
  add();
})


</script>
