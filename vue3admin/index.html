              # UI框架的选择

              # Ant design Vue

              Ant Design Vue官方文档open in new window
              这里我用的是 ant design vue 作为ui框架。当前版本："ant-design-vue": "^4.0.0"安装

              npm install ant-design-vue@4.x --save

              引入文件 在main.js中引入

              import Antd from 'ant-design-vue';
import 'ant-design-vue/dist/reset.css';

app.use(Antd)

              安装内置图标

              npm install --save @ant-design/icons-vue

              然后可以通过按需引入使用了

              import { StarOutlined, StarFilled, StarTwoTone } from '@ant-design/icons-vue';

<star-outlined />
<star-filled />
<star-two-tone two-tone-color="#eb2f96" />

              # Eement Plus

              Eement Plus 只能使用ts进行开发。
              ElementUI plusopen in new window
              当前版本： "element-plus": "^2.3.8"

              cnpm install element-plus --save
              # 全局引入
              import ElementPlus from 'element-plus'
import 'element-plus/dist/index.css'

app.use(ElementPlus)
