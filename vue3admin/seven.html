                            # 左侧菜单栏

                            处理根据登录接口传递的数据，进行左侧菜单栏的渲染。
                            我这里使用的 AntDesignVue4.0 框架进行开发。

                            # 动态菜单

                            主要是根据动态数据进行渲染菜单。

                            # 推荐使用

                            一般来说都使用这种 递归菜单 的写法。
                            也能用于多个层级的菜单展示。

                            逻辑梳理

                            1. 首先最外层写一个 a-menu 标签，传递一些菜单的属性，主题、默认选中key、是否打开收起
                            2. 新建一个文件，作为用于递归菜单的组件 命名 menu-tree ，并且引入使用，将菜单数据 items 传递下去
                            3. 构建 menu-tree.vue，编写递归组件的代码逻辑，具体代码看下面
                            4. menu-tree文件下，根据ant的规则，在 a-menu 标签下 使用 a-sub-menu 作为二级菜单
                            5. 循环遍历菜单数据，判断是否存在item.children（是否有二级菜单子菜单），来决定渲染逻辑。
                            6. 添加 需要的icon图标和点击事件（点击跳转等）

<template>
  <a-menu
    :inline-collapsed="state.collapsed"
    v-model:selectedKeys="state.selectedKeys"
    theme="dark"
    mode="inline">
    <menu-tree
      :menuList="items"
      @menuClick="menuClick" >
    </menu-tree>
  </a-menu>
</template>

                            menuTree.vue

                            <template>
    
    <template v-for="item in menuData" :key="item.key">
        <template v-if="item.children">
            <a-sub-menu :key="item.key">
                <template #title>
                    <i :class="item.icon"></i>
                    {{ item.label }}
                </template>
                <template v-if="!item.children">
                    <a-menu-item v-for="list in item.children" :key="list.key" @click="menuClick(list)">
                        {{ list.label }}
                    </a-menu-item>
                </template>
                
                <template v-if="item.children">
                    <menu-tree
                        :menuList="item.children"
                        @menuClick="menuClick" >
                    </menu-tree>
                </template>

            </a-sub-menu>
        </template>
        <template v-else>
            <a-menu-item :key="item.key" @click="menuClick(item)">
                <i :class="item.icon"></i>
                {{ item.label }}
            </a-menu-item>
        </template>
    </template>
    
</template>

<script setup>
import {reactive} from 'vue';

const emit = defineEmits(['menuClick']);

const props = defineProps(['menuList']);
console.log('props',props);

let menuData = reactive([]);
menuData=props.menuList;

const menuClick = (item) => {
  emit('menuClick', item);
};
</script>

                            # 菜单权限

                            通过登录接口返回的权限列表数据，渲染本账号觉得应该展示的菜单。

                            # 简单权限控制

                            这是一个比较简单的方法，后续会说到更好的办法。

                            1. 获取权限的数据列表
                            2. 通过 v-if对菜单栏的权限进行控制

                            总的就是通过判断权限列表中 是否存在该菜单的权限字段，来判断是否要展示这个菜单。

                            menuTree.vue
                            在一级菜单和二级菜单上，都添加了判断v-if="authBox.includes(list.auth)"

                            <template>
    
    <template v-for="item in menuData" :key="item.key">
        <template v-if="item.children">
            <a-sub-menu :key="item.key" v-if="authBox.includes(item.auth)">
                <template #title>
                    <i :class="item.icon"></i>
                    {{ item.label }}
                </template>
                <template v-if="!item.children">
                    <template v-for="list in item.children" :key="list.key">
                        <a-menu-item v-if="authBox.includes(list.auth)" :key="list.key"  @click="menuClick(list)">
                            {{ list.label }}{{ item.auth }}
                        </a-menu-item>
                    </template>

                </template>
                
                <template v-if="item.children">
                    <menu-tree
                        :menuList="item.children"
                        @menuClick="menuClick" >
                    </menu-tree>
                </template>

            </a-sub-menu>
        </template>
        <template v-else>
            <a-menu-item v-if="authBox.includes(item.auth)" :key="item.key" @click="menuClick(item)">
                <i :class="item.icon"></i>
                {{ item.label }}
            </a-menu-item>
        </template>
    </template>
    
</template>

<script setup>
import {reactive} from 'vue';

// 权限数据
let authBox = reactive(['home','user','auth','menuAuth','setting'])

const emit = defineEmits(['menuClick']);

const props = defineProps(['menuList']);
console.log('props',props);

let menuData = reactive([]);
menuData=props.menuList;

const menuClick = (item) => {
  emit('menuClick', item);
};
</script>
